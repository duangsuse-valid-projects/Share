    delim0/delim1
    
    
    指示后方加载字典表的 键值对/键-值 切分符，这可用于兼容 换行/= 外的格式
    
    
    style
    
    
    加载另外的 CSS 样式链接(':' 起始)，或者直接的文本
    
    
    conf
    
    
    加载符合 ?k=v 或 &k=v 格式的参数表，参数表内亦可用此参数另再加载
    
    
    text
    拼接主内容，可以 ':' 起始引用链接
    
    
    font-size
    渲染结果字体大小，比如 50px, 25pt
    
    
    mode
    使用词典名，无效则不变动
